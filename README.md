## 📖 Overview
This project analyzes IMDB movie reviews using Natural Language Processing (NLP) and machine learning.  
It predicts whether a review is **positive** or **negative**.

## 💼 Portfolio Impact
This project demonstrates:
- End-to-end **Data Science workflow** (EDA → preprocessing → modeling → evaluation).
- Application of **NLP techniques** (TF-IDF vectorization).
- Use of a **benchmark dataset (IMDB 50K reviews)** recognized in academia and industry.
- Clear documentation and reproducibility for recruiters and collaborators.
